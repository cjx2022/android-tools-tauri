================================================================================
                              ADB 放置说明
================================================================================

本程序需要 ADB (Android Debug Bridge) 工具才能正常工作。

【放置方法】

请将 adb.exe 文件放置到以下位置之一：

方法1（推荐）：
- 将 adb.exe 放在与本程序 (android-tools.exe) 相同的目录下
- 程序会优先检测同目录下的 adb.exe

方法2：
- 将 adb.exe 所在目录添加到系统的 PATH 环境变量中
- 程序会自动从 PATH 中查找 adb.exe

方法3：
- 首次运行时，如果程序未找到 adb.exe，会提示"未找到ADB，请手动设置"
- 点击"手动设置ADB"按钮，选择 adb.exe 文件的位置

【获取 ADB】

adb.exe 包含在 Android SDK Platform Tools 中，您可以从以下地址下载：
https://developer.android.com/studio/releases/platform-tools

或者从 Android Studio 的安装目录中找到：
<Android SDK 路径>\platform-tools\adb.exe

【验证 ADB 是否可用】

1. 打开命令提示符 (CMD)
2. 输入: adb version
3. 如果显示版本信息，说明 ADB 已正确安装

【注意事项】

- 请确保使用与您的系统架构匹配的 adb.exe（32位或64位）
- Windows 系统通常需要 adb.exe 以及相关的 DLL 文件
- 建议将整个 platform-tools 文件夹的内容一起复制到程序目录
