安卓工具集 v1.5.1
==================

一款功能强大的Android设备管理工具，基于 Rust + Tauri 技术栈开发。

功能特性：
----------
- 设备管理（USB/网络连接，支持批量操作）
- APK批量安装和卸载
- 显示控制（状态栏/导航栏，仅适用于安卓10及以下）
- 文件管理（上传/下载/浏览）
- APP配置备份与恢复
- ADB命令行终端
- 屏幕镜像（基于scrcpy）

使用方法：
----------
1. 双击"安卓工具集.exe"启动程序
2. 使用USB连接设备或通过网络连接（输入IP地址）
3. 设备列表支持全选功能，方便批量操作
4. 首次运行会自动创建.tools_cache文件夹存放内置工具

系统要求：
----------
- Windows 7/8/10/11
- 安卓设备需开启USB调试或网络调试

注意事项：
----------
- 删除操作不可恢复，请谨慎操作
- 显示控制功能仅适用于安卓10及以下版本
- 部分功能需要设备有root权限
